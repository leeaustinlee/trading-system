# 台股行情 Gate + 自我優化引擎

本文件是 Austin 台股系統的第一層交易 Gate。所有選股、進場、出場、追強之前，必須先判斷「今天是否值得交易」。若本 Gate 判斷不可交易，後續交易決策引擎不得輸出進場。

## 任務定位

第一任務不是選股，而是判斷：

```text
今天是否值得交易？
```

本文件必須先於 `D:\ai\stock\trade-decision-engine.md` 套用。

## 1. 行情等級判斷

行情等級只能選一個：

- A = 主升盤：可積極交易
- B = 強勢震盪：只做 1 檔，小倉
- C = 震盪 / 出貨盤：禁止交易

強制規則：

- 行情等級 = C：直接輸出「今日建議休息」
- 行情等級 = B：最多允許 1 檔，倉位 <= 3 成
- 行情等級 = A：才允許正常交易

## 2. 判斷依據

每次必須用 YES / NO 回答：

- 台積電是否有明確趨勢，非橫盤
- 主流族群是否一致，例如 PCB / 半導體 / AI 伺服器
- 強勢股是否續強，沒有集體轉弱
- 是否接近日高但未創高，形成高檔震盪
- 是否出現洗盤轉強，跌後拉回
- 是否出現爆量開高走低，出貨訊號

## 3. 行情階段判斷

只能選一個：

- 開盤洗盤期：09:00-09:15
- 主升發動期：最佳進場
- 高檔震盪期：風險高
- 出貨 / 鈍化期：禁止進場

## 4. 最終決策

只能選一個：

1. 可進場：列出股票 + 條件
2. 只可觀察
3. 今日建議休息

若行情等級為 C，最終決策必須是「今日建議休息」。

## 5. 可進場輸出要求

只有決策為「可進場」時，才可輸出最多 2 檔：

```text
【股票名稱】
進場：
停損：
停利：
倉位：
理由：
```

限制：

- 行情等級 A：可依資金規則正常交易，最多 2 檔
- 行情等級 B：最多 1 檔，倉位 <= 3 成
- 行情等級 C：禁止輸出可進場

## 6. 進階判斷

必須判斷：

- 是否為主升段
- 是否為洗盤轉強
- 是否為假突破
- 是否為資金換股

## 7. 風險警示

必填：

- 是否接近高點
- 是否量價背離
- 是否主流轉弱

## 8. 自我強化模組

每日 14:00 / 15:30 / 18:00 檢討時必須使用。

輸入：

- 今日盤型
- 今日是否交易
- 今日交易結果：盈虧
- 是否違反規則

輸出：

```text
📊 今日評分（0~100）
📉 錯誤原因（若有）
🔧 明日優化建議（1~2點）
```

評分原則：

- 遵守 C 級禁止交易：加分
- B 級只做 1 檔小倉：加分
- A 級有抓到主升段且未追高：加分
- 出貨盤仍進場：重大扣分
- 第一筆虧損後仍交易：重大扣分
- 未查現價或未做 10 分鐘方向比對：扣分
- 明確停損且有執行：加分

## 核心目標

- 先判斷「能不能交易」
- 再決定「要不要出手」
- 最後讓系統每天變強
- 禁止模糊回答，一定要給明確決策
